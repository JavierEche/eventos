Eventos Demo

Vistas sugeridas para consultar datos semilla de la base de datos.

Vistas creadas

  • vw_eventos_detalle: detalle completo con categoría y organizador.
  • vw_eventos_publicados: eventos visibles para landing.
  • vw_resumen_categoria: resumen por categoría.

Consultas demo

SELECT * FROM vw_eventos_publicados;
SELECT * FROM vw_eventos_detalle WHERE precio = 0;
SELECT * FROM vw_resumen_categoria;

Cómo ejecutar

  1. mysql -u root -p < demo_schema.sql
  2. mysql -u root -p < demo_seed_and_views.sql

Con esto tendrás estructura, datos base y vistas listas para demo.
